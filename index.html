What is DOM? üòç

DOM stands for Document Object Model. It is a programming interface for web documents, which provides a structured representation of the HTML or XML document. The DOM represents the page so that programs can change the document structure, style, and content. The DOM is commonly used with scripting languages like JavaScript to manipulate the content and behavior of web pages dynamically.
In simpler terms, the DOM is the way in which web browsers organize and represent web pages internally. It allows developers to access and manipulate the various elements of a web page, such as text, images, forms, and links, using scripting languages like JavaScript.

Js Built-in functions of the DOM üòç

Here are some built-in JavaScript functions of the DOM (Document Object Model):
getElementById(): This method is used to get a reference to a specific HTML element on the page by its unique ID.
getElementsByClassName(): This method is used to get a collection of HTML elements that have the same class name.
getElementsByTagName(): This method is used to get a collection of HTML elements that have the same tag name.
createElement(): This method is used to create a new HTML element.
appendChild(): This method is used to add a new child element to an existing HTML element.
removeChild(): This method is used to remove a child element from an existing HTML element.
setAttribute(): This method is used to set the value of an attribute on an HTML element.
getAttribute(): This method is used to get the value of an attribute on an HTML element.
innerHTML(): This property is used to get or set the HTML content inside an element.
addEventListener(): This method is used to add an event listener to an HTML element, so that a JavaScript function is executed when the specified event occurs.

More Js Buit-in Functions of DOM üòç

Sure, here are some additional built-in JavaScript functions of the DOM:
querySelector(): This method is used to get a reference to the first HTML element that matches a specified CSS selector.
querySelectorAll(): This method is used to get a collection of HTML elements that match a specified CSS selector.
classList: This property is used to get or set the class name(s) of an HTML element.
style: This property is used to get or set the inline style of an HTML element.
scrollIntoView(): This method is used to scroll the viewport to a specified HTML element.
getBoundingClientRect(): This method is used to get the size and position of an HTML element relative to the viewport.
requestAnimationFrame(): This method is used to request that an animation be performed, and then execute a JavaScript function when the next repaint occurs.
createElementNS(): This method is used to create a new SVG or MathML element.
getComputedStyle(): This method is used to get the computed style of an HTML element.
focus(): This method is used to give focus to an HTML element.
